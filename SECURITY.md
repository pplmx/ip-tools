# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by opening an issue at
https://github.com/pplmx/ip-tools/issues.

## Audit

This project uses `cargo audit` in CI and `cargo deny` for dependency
policy enforcement.
